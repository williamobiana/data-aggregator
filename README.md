# data-aggregator

Tool to aggregate financial data and store in database.

Retail sentiment data: 
* FXSSI

COT: cftc.gov

Economic data:
* twelvedata
* forexfactory (export JSON/XML, weekly calender for forecast & previous data)
* Trading economics: tradingeconomics.com
* Alphavantage
* Fxsheet: docs.fxsheet.com
* Investing.com
    * use python lib for api calls (import investpy as inv:)
